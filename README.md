# HalfKOS

HalfKOS is a preemptive operating system designed to run on Microcontrollers with 512 bytes of RAM.

## Design Goals

1. Low memory footprint

2. Simplicity of design

3. Expandability

4. Didactic code


## Features

1. Preemptive scheduler

2. Dynamic Task creation/destruction


## Memory Usage

### MSP430G2553

1. HalfKOS core

2. Task

3. Mutex

## Ports

1. MSP430G2553
